# مدیریت اعضا

این برنامه برای مدیریت اعضا طراحی شده است و امکانات مختلفی را ارائه می‌دهد.

## ویژگی‌ها
- افزودن و ویرایش مشخصات اعضا
- حذف اعضا
- مشاهده لیست اعضا
- جستجوی پیشرفته

## نصب
برای نصب این برنامه، ابتدا مخزن را کلون کنید:
```bash
git clone https://github.com/amir0933054-glitch/Am.git
```
سپس به دایرکتوری پروژه بروید و وابستگی‌ها را نصب کنید:
```bash
cd Am
npm install
```

## ساختار پروژه
این پروژه از ساختار زیر پیروی می‌کند:
```
Am/
├── src/
│   ├── components/
│   ├── screens/
│   └── utils/
└── package.json
```

## پایگاه داده
این برنامه از پایگاه داده SQLite برای ذخیره‌سازی اطلاعات اعضا استفاده می‌کند.

## ساخت APK برای اندروید
برای ساخت APK، از دستور زیر استفاده کنید:
```bash
npm run build
npm run android
```
